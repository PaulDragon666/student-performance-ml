# 🎓 Student Performance Prediction (ML Project)

## 📌 Overview
This project predicts students' math scores using machine learning.

---

## 🚀 Key Highlights
- Feature engineering improved R² from ~0.23 → ~0.87
- Ridge Regression performed best
- Strong linear relationship observed

---

## 🧠 Feature Engineering
Avg Score = (Reading + Writing) / 2

---

## 🤖 Models Used
- Linear Regression
- Ridge Regression (Best)
- Random Forest, Gradient Boosting
- XGBoost, LightGBM, CatBoost

---

## 📈 Final Results
- R² Score: ~0.87
- RMSE: ~5.4

---

## 🔍 Key Insights
- Avg Score is the strongest predictor
- Economic condition affects performance
- Feature engineering matters more than model complexity

---

## 📂 Files
- Notebook (.ipynb)
- Trained model (.pkl)

---

Install dependencies and open notebook.
